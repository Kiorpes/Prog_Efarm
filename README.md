# Prog_Efarm project for PHP & MySQL

Installation Steps:

1. Copy the 'Prog_Efarm_22038' folder into htdocs
2. Make sure both Apache & MySQL are active in XAMPP
3. Create a Database named "Hotel" in phpmyadmin
4. Head to http://localhost/Prog_efarm_22038/Set-Up/ 
(Note: If you named your database something else than "Hotel" or prefer not to use the 'root' user for mysql, change those settings in the tiny db.php folder)

5. Run the Creation and Insertion files. The numbers on their names indicate the order in which to run them.
6. Head to http://localhost/Prog_efarm_22038/Program/
(Note: If you named your database something else than "Hotel" or prefer not to use the 'root' user for mysql, change those settings in the tiny db.php folder)

That will complete the installation!

To access the customer page, enter http://localhost/Prog_efarm_22038/Program/Entry_Page.php and browse at your leisure.
To access the admin pages, enter http://localhost/Prog_efarm_22038/Program/Admin_Login.php 
If you ran the commands in step 5, there will be two admins: Username Vasilis Password kior and Username kapoios password allos
